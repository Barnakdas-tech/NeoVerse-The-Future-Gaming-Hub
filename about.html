About NeoArcade

A Cyberpunk-Inspired Gaming Interface

Our Story

NeoArcade is a cyberpunk-inspired gaming website created as part of the SNW Capstone Project under the mentorship of Ajay Sharma sir. The website draws inspiration from neon-lit futuristic worlds, advanced interfaces, and immersive gaming environments.

What This Project Represents

This project showcases my understanding of core web development technologies: HTML for structure, CSS for styling, and (upcoming) JavaScript for dynamic effects and interactivity.

Vision

NeoArcade aims to deliver a stunning futuristic UI while demonstrating creativity, design skills, and fundamentals of modern web development.

Future Enhancements

  • JavaScript-based animations & UI transitions
  • Interactive menus and components
  • Dynamic content loading
  • Futuristic hover, glitch, and neon effects

Project Details

Student: Barnak Das

Subject: SNW (Software for the New World)

Mentor: Ajay Sharma

Theme: Cyberpunk Games

Phase 1: Static Website (HTML + CSS)

Phase 2: Dynamic Website (JavaScript)

Why I Built This

Passion for Games

I began this project because I’ve always been fascinated by futuristic games, neon visuals, and cyberpunk aesthetics. I wanted to build something that reflects the world I imagine while gaming.

Learning Web Development

The SNW subject gave me a structured path to learn HTML, CSS, and JavaScript. This project is my way of applying those skills into a real, creative website.

Create Something Unique

I didn’t want to build a normal website — I wanted something bold, glowing, futuristic, and visually different from standard projects. NeoArcade is my attempt to design a UI that feels alive.

Capstone Evaluation

This project is also my official Capstone submission for Phase 1 and Phase 2. It shows my growth from static layouts to dynamic interactive features.

A Step Toward the Future

NeoArcade is more than a project — it is my stepping stone into UI design, front-end development, and creative futuristic interfaces.
